## Локальная установка

```
# Клонировать проект
git clone git@github.com:Evgen-Polyanskii/backend-project-lvl4.git

# Перейти в директорию проекта
cd backend-project-lvl4

# Установка
make setup
```

## Запуск

```
make start
```
Открыть в браузере страницу http://localhost:5000
